# TypeSpec API Specifications

This directory contains TypeSpec definitions for the Travel Planner API.

## Overview

TypeSpec is used to define API contracts in a type-safe manner. These definitions automatically generate:
- OpenAPI 3.0 specification (`public/api/openapi.yaml`)
- JSON Schemas for validation (planned)
- TypeScript types for frontend (planned)

## Commands

```bash
# Compile TypeSpec definitions
npm run tsp:compile

# Watch for changes and auto-compile
npm run tsp:watch

# Format TypeSpec files
npm run tsp:format
```
